# Padrão de Estilo com TailwindCSS

## Cores principais
- Primária: `bg-white`, `text-dak`
- Secundária: `#3976bd`, `text-white`
- Feedback: `green-600`(sucesso), `red-600`(erro), `yellow-500`(aviso)

## Tipografia
- Título: `text-xl font-bold text-gray-900`
- Subtítulo: `text-lg font-semibold text-gray-700`
- Texo: `text-base text-gray-600`

## Botões
- Botão Primário

```html
<x-primary-button />
```
